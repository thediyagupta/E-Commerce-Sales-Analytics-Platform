#  E-Commerce Sales Analytics Platform

A full-stack analytics platform built on the Olist Brazilian E-Commerce dataset using **PostgreSQL, FastAPI, Redis, Docker, and Python**.

The platform exposes SQL-powered REST APIs for business intelligence and visualizes insights through an interactive dashboard.

---

## Features

- 📈 Monthly Revenue Trends
- 👥 Cohort Retention Analysis
- 🎯 RFM Customer Segmentation
- 🚚 Delivery Delay vs Review Analysis
- 🏆 Top Sellers by Region
- ⚡ Redis Query Caching
- 🐳 Dockerized PostgreSQL & Redis
- 📊 Interactive Dashboard
